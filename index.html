My Projects

Cocoket

Developed an application that connects coconut sellers with bricket manufacturers and people in need of brickets. The application helps coconut sellers get value out of unused coconut shells. It also helps manufacturers find raw materials (coconut shells and fibers) and makes it easier for the general public to get cooking fuel.

Fingerprint Checker

Implemented a biometric-based individual detection system desktop app (C#) using fingerprint images, employing pattern matching techniques like the Knuth-Morris-Pratt (KMP) algorithm, Boyer-Moore (BM) algorithm, and Regular Expressions (Regex) for pattern matching tasks.

SeTiket

Created the backend for SeTiket, a platform designed to streamline the management of events and ticketing processes. SeTiket offers users the ability to discover, book, and manage events efficiently while providing organizers with tools to handle ticket sales and notifications effectively.
